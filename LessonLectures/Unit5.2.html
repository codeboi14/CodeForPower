Unit 5.2: Populating Arrays & For Each Loop

How Are Arrays Populated?

  1) You can populate an array with literals (like we learned previously)

    int[] myArray = {0,1,2};

  2) You can populate an array with loops (typically for loop).

    Populating an array with a for loop is better than just listing what elements it contains.
    Here's why: consider declaring an array holding int values from 0 to 500 inclusive.
    Yea...good luck writing out 501 integers with the listing literals method. Using a loop will be much easier.




For-Each Loop (Enhanced For Loop)
The for-each loop, or enhanced for loop, is simply another way to write the traditional for loop but under more specific circumstances. The enhanced for loop is specifically made to loop through collections of data, like an array.

 Syntax: 
    for (initialized LCV declaration : arrayName){
         //code block
    }
 Example: 
    String[ ] colors = {"blue", "green", "yellow"};
    for (String i : colors) {
         System.out.print(i + " ");
    } 
    //outputs: blue green yellow
Breakdown:
 1) The String "i" holds elements of the colors array.
  Instead of using a traditional for loop to call elements like: colors[i], you just write "i" in the for each loop.
 2) The for each loop can be read like this:
   for each String element called i -as in the index of the array- in colors, print out the value of i.

Comparison Between The For & For-Each Loop:
When looping through an array, the for-each loop is much shorter than a for-loop.
Given an array called colors, lets say we want to write a loop that prints out all the elements of the array.

Given: String[ ] colors = {"blue", "green", "yellow"};

for-each (enhanced for loop) 
	for (String i : colors) 
	{
		System.out.print(i + " ");
	}
for-loop 
	for(int i = 0; i < colors.length; i++)
	{
		System.out.print(colors[i] + " ");
	}
The two loops all accomplish the same thing, outputting: blue green yellow


Final Thing To Know About Enhanced For Loops:

 1) You CANNOT use the enhanced for loop for non-objects in Java.

 2) Because you don't write any increment/decrement statement in the enhanced for loop,
  it can ONLY start from the beginning of an array and go to the end.
  On the other hand, traditional for loops can go both ways.
