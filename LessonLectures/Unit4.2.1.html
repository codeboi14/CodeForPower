The while loop The while loop is the most general of the three types of loops in Java, and it is declared with the keyword “while”. ALL loops in Java can be written into a while loop. Like the for loop, the while loop continues a repetitive action until a specific boolean condition is reached. You should use the while loop when you don’t know how many times something will loop.

Syntax: 
while (boolean condition check) {
    //code
    //increment/decrement
}

The do-while loop
The do-while loop follows a slightly different execution sequence than the while loop. The do-while loop checks the boolean condition AFTER executing its code block, contrasting the for and while loop, which checks the boolean condition BEFORE. It is because of this that do-while loops are guaranteed to execute at least once. The keywords “do” and “while” are necessary to declare the do-while loop.

Syntax: 
do {
    //code 
    //increment/decrement
}
while (condition, boolean);

2) Check the condition (LCV is updated inside “do”)

- (if false) → Break the loop

Do-while loops are also called Exit Control Loops, as they exit loops to check for the control (use see this with the separate code (inside “do”) and boolean condition statement (inside “while)).
