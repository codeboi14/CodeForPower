Unit 3.1: Boolean Expressions

Booleans:
When programming, you often need a data type that can only have one of two values, like:
   TRUE / FALSE
   ON / OFF
   etc.
These conditional statements can help accomplish numerous things. For example, you want to program an algorithm to buy a product. You can say if something is over $10, don't buy it. If something is under $10, buy it.

The boolean data type handles conditions in Java, as it can only store true or false.

Conditional Statements
Conditional statements control the execution of code based on the true/false return from a boolean expression.
There are two conditional statements in Java; the if and else conditional statements.

 - Use if to specify a block of code to be executed, if a specified condition is true
 - Use else to specify a block of code to be executed, if the same condition is false
 - Use else if to specify a new condition to test, if the first condition is false
