Unit 5.4: ArrayLists

What Is An ArrayList?
An ArrayList is a resizable array, meaning that it doesn't contain a predetermined size, and can contain dynamic input. Additionally, ArrayLists can store objects ONLY, and come with its own set of special methods used to perform basic functions, like adding, acessing, & removing elements of the array.

ArrayList Syntax:

  ArrayList <yourObj> identifier = new ArrayList <yourObj>();

1) Specify the class name: ArrayList
2) Provide the object type you want the ArrayList to store in <yourObj>
3) Give you arrayList an identifier(name).
4) Write the "new" keyword, the class ArrayList, and it's parameters (just like declaring an object).

Example:

  ArrayList <String> myArrayList = new ArrayList <String>();


Important Aspects About ArrayList:

1) It's Resizeable: You know that plain arrays have a predetermined size, meaning that you can't add or subtract elements from the array. Since ArrayLists are dynamic, they are able to overcome this issue.

2) Storing Objects: Because ArrayLists can ONLY store objects, you can't store primitives like int, double, boolean etc. To overcome this issue, wrapper classes that "wrap" around the primitive in a class structure are provided. Note that wrapper classes basically act like regular primitives, so don't worry how they function. You just need to know how to declare them.

For example, to store an ArrayList of int type, specify its wrapper class instead.

  ArrayList <Integer> myArrayList = new ArrayList<Integer>();
