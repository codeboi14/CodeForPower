Unit 4.3: Nested Loops

Nested loops are loops inside of loops, and to get to the innermost nested loop, one must first go through all the outermost nested loops. Note that all loops (while, for & do-while) can be nested.

Nested loops can be quite confusing logically, as you have so many loops to worry about. However, if you carefully trace the loop step by step, you will soon see a pattern!
Read the following example carefully. If you don't understand the logic behind it, plug it into Eclipse and test it for yourself. Remember that coding is a hands on, and one will never master it without actually trying it out!

Nested Loop Example I.

 

Tracing The Loop:
Trace the loop from the inside out.

  1) Loop3's code block will print "*". The loop continues this process for 4 times.

  2) Loop2 has a code block including loop 3 and a println statement with the symbol "!".
   After loop 3 finishes in its entirety, the "!" is printed at the end of loop3's "****".
   Note that loop2 continues for 3 times.

  3) Loop 1 has a code block containing loop2 and a println statement printing "-----".
   After loop2 finishes in its entirety, the "-----" statement is printed underneath.
   The loop continues this process for 2 times. After all this finishes, we arrive at our output.

The Big Takeaway:
In nested loops, the innerloop in its entirety is one code segment of the outer loop, which will repeat its code block for generically, n, times. Because of this, the number of iterations can be quite lengthy. However, you should see a pattern form after a few passes.

Tracing The Loop:
Like the previous example, trace the loop from the inside out.

  1) The inner loop will have its iterations decrease through every loop. First, it will print "*" 5 times.
  2) The outer loop finishes its code segment, and is updated by one, so i = 1.
   The j value in the second loop changes by: 5 - i = 4, so the loop iterates 4 times, printing "*" 4 times.
  3) The same logic keeps repeating until j goes to 1, and then, the loop terminates.
   Hence, this reverse right triangle like shape is printed.
