The else statement
Use the else statement to specify a block of code to be executed if the condition is false. Unlike the if statement, an else statement cannot exist on its own, and must come in pairs. If you think about it, this makes sense, since you must have an if statement in the first place so that the else condition might happen!

Syntax:


Do you see that there is no boolean condition in the else statement?
Else is simply just what happens if the if statement is not true.


else if
Else statements can be paired with more if statements, but not more else statements. These are called "else if" statements. Use else if to specify a new condition if the first condition is false.



Here is a real-world-ish example of using else if:
Let's say you want to write code telling a robot to steal something only if the reward is greater than the risk.
(We don't encourage stealing! Computer science is a honest job, but this is just a light-hearted educational example).

You might write code with the logic:

 If the reward is above $100. Steal.

 Else if the reward isn't $100, see if the security is low. If yes, steal.

 Else, don't steal.

Now you see why else if is useful!
