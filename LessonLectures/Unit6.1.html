Unit 6.1: Linear Search

Linear/Sequential Search
A linear search, also known as a sequential search, is pretty easy. Loop through the entire array until you find your target.



You can probably tell how straightforward linear searches are. Just use a for loop that loops through the entire array, while checking each value for the target. However, linear searches aren't really efficient. Think about it like this, let's say you want to flip to page 450 in a large textbook. Linear search will find page 450 by checking page 1, then page 2, etc. In extreme cases when your program is very long, doing linear searches can time-out in programming competitions, and that's a big no-no in computer science.
