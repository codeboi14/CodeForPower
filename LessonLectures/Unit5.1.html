Unit 5.1: Arrays

What Are Arrays?
Arrays are a type of data structure that can store a large amount of data in a single place. Each data structure comes with its own benefits and disadvantages in terms of usability under different circumstances. Having said that, arrays will be useful when there is a fixed amount of data input, and not so useful when there is dynamic data input. The last thing to know about arrays is that they are always objects.

How Are Arrays Declared in Java?
Arrays are declared with the variable type that you want the array to store, followed by square brackets. The variable type can be a primitive or an object from the Java API or a user defined class. Note that an array can ONLY store the data type that it was assigned to in its declaration.

Note* If an array is declared but not assigned any value within itself, numeric data types will be defaulted to 0, booleans will be defaulted to false, and objects (reference types) to null.

 Syntax:
   1) There are two ways to declare an array. The first method doesn't assign any values to the array,
    but just declares that it exists, and can store sizeOfArray amount of values.

     dataType[] arrayName = new dataType[sizeOfArray];

   Example:
     int[] num1 = new int [3];

      //this array called num1 can hold 3 integers. (These integers have not been assigned).


   2) The second declaration method assigns values to the array right away.
   The array's index (or how many values it can hold) is the number of values that you list in the array.

     dataType[] arrayName = {instances of dataType1, instances of dataType2...instances of dataTypeN};

   Example:
     int[] num2 = {1, 2, 3};

     //This array called num2 stores integers 1, 2, 3.


In-Depth Recap:
The two sides of the assignment symbol (=) signify two steps in the process of declaring an array. The left side tells the Java Compiler that the name of your array will hold an array of your chosen data type. The right side is the actual declaration of the array, signified by the "new" keyword. This assigns the arrays to a REFERENCE location in memory, in which the contents of the array are stored. Due to these two steps, Java arrays are dynamically allocated. Furthermore, arrays have a predetermined size that you give it when its declared, which dictates how many elements it can store.

Array Length
You can find the length of an array by using the length method.
IMPORTANT! Unlike strings, the array's length method doesn't have parenthesis! 
Arrays:
 String myPets = {"dog", "cat", "hamster"};
 myPets.length; //gives the length/size of the myPets array. 

Strings:
 String myString = "hi";
 myString.length(); //gives the length/size of the myString String.
