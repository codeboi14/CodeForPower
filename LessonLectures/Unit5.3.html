Unit 5.3: 2-D Arrays

Array Dimensions
Arrays are quite special, as they can have as many dimensions as you want. The dimension of an array is the number of indices that are necessary for the specification of an element in the array. Furthermore, the amount of dimensions an array has can be identified by the number of square brackets [] that follow the data type of the array.



Note* One bracket is known as a linear array while N brackets (N > 1) are called N-dimensional arrays.

Interpreting Multi-dimensional Arrays
Multi-dimensional arrays are arrays inside of arrays. This concept can be difficult to grasp, so we will provide some ways to think about them. 
Real World Example:
A vending machine has rows and collumns to show where different goodie-goods are located. For the rows, it has an array storing that data. For the collumns, it has another array storing that data. Now that's a 2D array! Let's say you want to access the "Lip-Popping Smack Gum". Well, you have to specify the row FOLLOWED by the collumn that the gum is located in so you can get it out of the vending machine. Thus, you might do something like this: giveMeStuffIn[0][1]. Now you just learned how to access elements in a 2D array! Wow!

 Syntax (2-D array):
  1) Declaring an array with N rows and M columns:
   int[ ][ ] num2 = new int [Nrows][Mcollumns];

  2) Directly declaring an array with curly braces:
   int[ ][ ] num3 = {{1, 2, 3... N-2, N-1}, {1, 2, 3... M-2, M-1}};

Given the 2-D Array, You Can Visualize It Like This:

3-D Arrays
Just in case you encounter 3-D arrays in your computer science class, here is something that you should know. 3-D arrays are basically extensions of 2D arrays. You can think of 3-D arrays as an x-y-z plane, useful for volume (length, width, depth). 

Syntax (3-D array):
int[ ][ ][ ] num2 = new int [N][M][K]; 

Declaring a 3-D array with curly braces:
int[ ][ ][ ] num3 = {{1, 2, 3... N-2, N-1}, {1, 2, 3... M-2, M-1}, {1, 2, 3... K-2, K-1}}; 

Example (3-D array):
int[ ][ ][ ] num2 = new int [2][3][4];
int[ ][ ][ ] num3 = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
