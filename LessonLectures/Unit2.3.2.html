Unit 2.3.2: Constructors

A constructor is a special method that initializes objects.
It is called when the object of a class is created, and used to set values for object attributes.

How To Create An Constructor
 1) The constructor name MUST match the class name
 2) The constructor MUST not have a return type (like void etc.)
 3) If you don't create a constructor, Java creates a default one for you.
  A constructor can take in as many parameters as you want.

Variable Scope
Everything in Java has a scope, meaning when it is in the memory. If a variable is out of scope, it is out of the computer's memory, and you can't use it.

Scope in Methods, Classes & Constructors
The scope for variables in methods & classes are inside the curly braces {}. They are also called "local variables".

Example: If you have a local variable in a method, you can't use it outside of the method, since its scope is only inside that method. (This same logic applies to classes as well).
