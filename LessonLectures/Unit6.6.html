Unit 6.6: Merge Sort

Merge Sort:
The merge sort is probably the most challenging algorithm that you need to know for the AP Computer Science A exam. Merge sort runs in O(NlogN) time as it splits a data set into two data sets (logN time) and merges the two sorted sets together in linear time (N time).

The merge sort is a divide and conquer styled algorithm, where each data set is split into groups of two until it is split into its most basic elements (arrays of size 1). From there, each small piece is merged together in order. Below is a diagram illustrating this concept. Just analyze the LOGIC and big picture behind merge sort. Don't worry about coding it for now.
