ASCII (Data Representation)
All data stored in a computer must be represented (or encoded) as bits, 0's and 1's. Standard codes (ASCII and Unicode) were established so that data (letters, numbers, punctuation etc.) could be exchanged among different kinds of computers easily.

ASCII

ASCII is short for The American Standard Code for Information Interchange.

ASCII characters/format that you should memorize:

 32 stands for (Space or Blank)
 48..57 → '0'..'9' (digits)
 65..90 → 'A'..'Z' (UPPERCASE LETTERS)
 97..122 → 'a'..'z' (lowercase letters)
 (others ASCII values represent punctuation of "special" characters)

NOTE: ASCII is 8-bit code, meaning that there are 256 (2^8) different ASCII characters. This is enough for the English alphabet, but it might not be enough for other languages like Chinese. Hence, unicode needs to be used to represent those characters (16-bit code).

Unicode
Unicode is the successor to ASCII, with 16 bits or 2^16 (65,536) possible combinations, enough for all the known languages.

Note* The first 256 Unicode chars are the ASCII chars.
Summary: all characters, letters, digits, and punctuation are stored as ASCII/Unicode digits.
