Unit 4.1: For Loop

What Is A Java Loop?
As the name implies, a loop is a section of code that repeats a certain action for a certain amount of time. There are three types of loops to learn in Java: the do-while, for, and while loop. These three loops are all slight variations of one another.

Requisites for All Loops (The 3 C's)
All loops require the following conditions in order to function properly:

 1) Counter initialization (loop control variable, LCV)
 2) Conditional check (a boolean that deternmines when to stop the loop)
 3) Counter update of the LCV (increment/decrement)

The 3 C's within any loop serves to limit how many times the loop should iterate, and update certain values. Without these conditions in place, the loop will not compile or result in an infinite loop (a run-time error). Note that these three components may appear in different places within the while and do-while loops due to logical necessity.
The for loop is the exception to this as it requires all three components to be listed according to its syntax.

How Does A Java Loop Function?
The following sequence of execution is consistent throughout all three types of loops in Java.

 1) The LCV is initialized.
 2) The LCV is checked with the boolean condition to see if it returns true.
   - (if true) → The code within the loop is executed
   - (if false) → The loop stops executing
 3) The LCV is updated. Go back to #2.


The for loop
Thanks to its clear syntax layout, the for loop is the easiest iterative statement to understand. Note that the keyword "for" is necessary for Java to identify the loop as a for loop.

Syntax:

  for (initialized LCV declaration; boolean statement; incrementer/decrementer)

Basic Things about Loops

 1) You can have whatever code you want inside a loop, such as more loops!
  This is called nested loops (more on this topic later).

 2) The curly braces {} are not necessary.
  Should they not be present, only the FIRST statement underneath the loop will be controlled by the loop.

 3) Loops are not methods.

Why For Loop Rocks!
The for loop is the easiest loop to debug and understand, so prevalent usage of this type of this loop is highly recommended.

Here is an example why: you use while loops when you don't know how many times something will loop. You can kind of think about this like... the Youtube algorithm. If the algorithm works in helping you find interesting videos, that's great! If it doesn't, then...you are kind of stuck hopeless. When using the for loop, you are in charge!

Beware of The Infinte Loop!
The main precaution you must take when programming loops is to avoid infinte looping, which happens when your boolean condition fails to stop the loop, hence making it run indefintley. For example, you want to want to write a loop that prints out digits from 0 to 1000 through incrementing. If you set your boolean condition to terminate the loop when the digit reaches -1000, obviously this will never happen, so the loop will continue indefintley.
