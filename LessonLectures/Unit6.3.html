Bubble Sort
Bubble sort is an easy sorting algorithm that is useful for numerous applications in Java. Bubble sort basically swaps each variable in an array chronologically if the adjacent elements are unsorted. Here is how it works:

 1) The elements at index 0 & 1 are compared. If the values are out of order, swap.
 2) The elements at index 1 & 2 are compared. If the values are out of order, swap.
 3) This process continues until it reaches the end of the array.
  If the whole array isn't sorted, complete the entire process again.
  Keep doing this until the array is sorted.

Note that if NO SWAPS are made after checking each adjacent element in the array, bubble sort knows that the array is in order, and exits the loop.

Did you see how each element in the array slowly "bubbled up" and "sank down" after each pass in the sort? In our context from sorting least to greatest, bubbling up refers to the greatest value going up to the greatest index value (end of the array), while the smallest value of the data set sinks down to the lowest index value (beginning of the array).

Note that Bubble sort runs in O(N) squared time in the worst case, meaning that it is EXTREMELY slow. You can even see this in our example, where many comparisons were "unnecessary" as no swaps happen.
