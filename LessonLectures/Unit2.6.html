Unit 2.6: JAVA Packages

Packages are used to group related classes, so it is another step of organization. Think of them like a folder containing files in your computer, or like the image below:



Benefits Packages Provide:
 1) Packages help avoid name conflicts.
  One class name in package1 won't conflict with the same class name in package2.
 2) Packages make your code organized and maintainable.

Types of Packages
There are two types of packages:

 1) Built in packages from the JAVA API (or Java library).
 2) User defined packages.

Packages From The API:
The Java API contains numerous prewritten classes for you to use.
You can import whole packages or an individual class.



User defined packages:
You can create your own packages to organize your classes!
To do this, you will need to understand the Java file system & directory.

*We won't cover these package types since they aren't tested on the AP Computer Science A exam.
