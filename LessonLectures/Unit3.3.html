DeMorgan's Laws
DeMorgan's Laws states that:

 !(A||B) = !A && !B → not(a or b) = not a & not b
 !(A&&B) = !A || !B → not(a and b) = not a or not b

These "truths" are really fundamental to understanding conditional statements, and should be memorized.

Logical Operators and Truth Tables
Now that you have a taste on what conditional statements are, we can introduce a new concept: truth tables.
The purpose of a truth table is to help determine whether a complex conditional statement is true (1) or false (0).

For example, you can use a truth table to prove DeMorgan's Laws:
 !(A||B) = !A && !B
 !(A&&B) = !A || !B

See how the truth table breaks the conditional expressions into "bite sized" pieces so that the end result is easy to calculate.


Note that for A and B initially, every possible combination between truth and false is listed.
This is so that all conditions going forth in the truth table can be calculated.
You can think that these combinations of 1s and 0s are like a firm "anchor".

Truth tables are your tools to help figure out complex conditional statements, so use them!

Summary:
  - &&, if at least one of the two booleans is false, then the whole expression is false.
  - ||, if at least one of the two booleans is true, then the whole expression is true.
  - !, reverses conditions.
  - Very important relations between boolean expressions, are reflected within DeMorgan's laws.
